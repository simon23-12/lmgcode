Über LMG Code / Wissenswertes

Dieses Programm wird von Herr Fuhs entwickelt und dient dazu, Schülerinnen und Schüler vom Leibniz-Montessori-Gymnasium Düsseldorf beim Erlernen von Agentic Coding zu unterstützen.

Lernmodus

Im Lernmodus gibt der Assistent fertigen Code aus, erklärt aber dabei Konzepte und gibt gezielte Hinweise — so wie ein guter Lehrer. Der Modus lässt sich mit /learn oder dem Schalter im Chat-Header ein- und ausschalten. Er ist pro Agent separat einstellbar.

Modellanbindung

Die Infrastruktur für Modelle ändert sich häufig und wird häufiges Anpassen dieser Seite notwendig machen. Die Anbindungen laufen über die Freikontingente von Google AI Studio (Link) und OpenRouter (Link).

Datenschutz

Eingaben werden an die jeweiligen Modell-Provider (Google/OpenRouter) gesendet, um die Antwort zu generieren. Bitte gebt keine privaten Daten oder Klarnamen in den Chat ein.

Tech Stack

  • Frontend: Vanilla HTML/CSS/JS, keine Frameworks, kein Build-Step
  • Editor: Monaco Editor 0.47.0 via jsDelivr CDN (AMD-Loader)
  • ZIP-Entpacken: fflate 0.8.2 via CDN
  • Markdown-Rendering: marked.js 12.0.0 via CDN (nur Bot-Antworten)
  • Icons: Font Awesome 6.5.0 via CDN
  • Backend: Vercel Serverless Functions (Node.js, ESM)
  • Code-Execution: Pyodide 0.27.0 (Python, WASM im Browser) · Wandbox API (C, C++, C#, TypeScript, Ruby) · JDoodle API (Java)
  • AI-Modelle: Sechs Modelle, User wählt im Dropdown — Standard: Gemini 3.1 Flash Lite
    • gemini-3.1-flash-lite-preview — Gemini 3.1 Flash Lite (Standard) via Google AI Studio
    • meta-llama/llama-3.3-70b-versatile — Llama 3.3 70B via Groq
    • qwen/qwen3.6-plus:free — Qwen via OpenRouter
    • stepfun/step-3.5-flash:free — Step 3.5 Flash via OpenRouter
    • nvidia/nemotron-3-super-120b-a12b:free — Nemotron 3 Super via OpenRouter
    • gemma-4-31b-it — Gemma 4 31B via Google AI Studio
